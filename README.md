## Description

Brain Games -- набор консольных мини-игр для тренировки логики и математики. Нужно дать 3 правильных ответа подряд.

## Requirements

- Python 3.13+
- uv

## Installation

    git clone https://github.com/moosechoose/python-project-49.git
    cd python-project-49
    make install

## Usage

    uv run brain-even
    uv run brain-calc
    uv run brain-gcd
    uv run brain-progression
    uv run brain-prime
